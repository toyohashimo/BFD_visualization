## ℹ️ このディレクトリについて

このディレクトリには、各機能の実装過程で作成された詳細な記録が保管されています。
日常的な利用には必要ありませんが、以下の場合に参照価値があります：

- 実装の背景や意思決定の経緯を知りたい場合
- 類似機能を実装する際の参考資料として
- プロジェクトの歴史を振り返る場合

現行の機能説明については、[02_features/](../02_features/) を参照してください。

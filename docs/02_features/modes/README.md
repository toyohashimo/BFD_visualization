# 分析モード

BFD Analyticsは全23の分析モードを提供します。

## 📊 モード分類

### [detailed_analysis/](detailed_analysis/) - 詳細分析モード (14種類)
単一Excelファイルから複数のブランド/セグメントを選択して横断比較

**カテゴリ**:
- Mode 01-03: ファネル分析①
- Mode 04-06: ファネル分析②（時系列）
- Mode 07-08: ブランドイメージ分析
- Mode 09-10: ブランドパワー分析（現在パワー）
- Mode 11-12: 将来性パワー分析
- Mode 13-14: アーキタイプ分析

### [historical_comparison/](historical_comparison/) - 過去比較モード (9種類)
最大3つのExcelファイルを読み込み、時系列でのデータ推移を分析

**カテゴリ**:
- Mode 01-02: ファネル分析（過去比較）
- Mode 03: ブランドイメージ分析（過去比較）
- Mode 04: ブランドパワー分析（過去比較）
- Mode 05: 将来性パワー分析（過去比較）
- Mode 06-09: ブランド軸過去比較（実装済み）

## 🎯 目的別モード選択

### 競合比較
→ 詳細分析 Mode 01, 04, 07, 09, 11, 13

### セグメント別分析
→ 詳細分析 Mode 02, 05, 08, 10, 12, 14

### トレンド分析
→ 過去比較 Mode 01-09

### ブランドイメージ
→ Mode 07-08 (詳細), Mode 03, 08 (過去比較)

### ブランドパワー
→ Mode 09-12 (詳細), Mode 04-05 (過去比較)

### パーソナリティ分析
→ Mode 13-14 (詳細), Mode 06 (過去比較)
